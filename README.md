# Buy vs Rent Calculator

A UK property calculator that compares the long-term financial outcome of buying a property versus continuing to rent.

## Features

- Side-by-side buy vs rent wealth projection over a configurable time horizon
- UK Stamp Duty (SDLT) calculation with first-time buyer, home mover, and additional property rates (post-April 2025)
- Separate cash savings and investment pots with different growth rates
- Monthly budget stress test with affordability traffic lights
- All calculations run client-side — no data leaves your browser

## Development

```bash
npm install
npm run dev       # http://localhost:5173
npm test          # unit tests
npm run build     # production build
```

## Stack

React, TypeScript, Vite, Tailwind CSS, Recharts.
